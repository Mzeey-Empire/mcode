V1 · Split Header

The disclosure row keeps its single-click expand behaviour — that part is fine. The fix is to give per-file actions their own row: a thin toolbar that appears between the file header and the diff content when the file is open. Every action is a real <button>, never nested.

  • DisclosureWhole row expands on click
  • ActionsSeparate toolbar, real buttons
  • Hit target26 × 26 px minimum
  • Tab orderChevron → name → toolbar → diff
  • KeyboardEnter/Space toggles; Tab walks toolbar
  • DOMNo nested interactives, no stopProp
The key idea: the file header has one job (toggle). The toolbar has one job (act). Misclicking the toolbar never collapses the row, and misclicking the row never fires a per-file action.
