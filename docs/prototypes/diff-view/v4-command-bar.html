V4 · Command Bar

The boldest take: file rows stay perfectly clean — chevron, name, stats, nothing else — and a small floating command bar slides in when the row is hovered or keyboard-focused. The moment the row expands, the bar pins, so it stays reachable while reading. Every action shows its single-letter shortcut, turning the diff panel into a keyboard surface.

  • RevealHover or focus-within
  • PinStays when expanded
  • KeyboardP preview · C copy · O open · Esc
  • Hit target26 × ≥30 px per command
  • DOMBar is a sibling, never inside the row
  • Trade-offMost novel — needs onboarding
The key idea: commands belong to the file, not the row. The bar is a sibling element layered on top of the row's right edge — never a descendant of the disclosure button. The pinned behaviour when expanded means you never lose actions while reading.
